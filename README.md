# photos-alextoop-com

## About
This is a simple website to test out amplify

## Prerequisites
Requires global vue cli and amplify cli.
```
npm install -g @vue/cli @aws-amplify/cli
```

## Project setup
```
npm install
```

## publish
```
amplify publish
```

## Compile and hot-reload for development
```
npm run serve
```

## Lints and fixes files
```
npm run lint
```
